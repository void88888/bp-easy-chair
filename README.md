# 💺 EasyChair - BP Deliberation Tool

A web app to simplify British Parliamentary (BP) debate chairing!

## ✨ Features
- Log judges' calls
- Mark interchangeable exchanges 🟡
- Auto-rank exchanges by resolution difficulty (easy → hard)
- Visual final consensus call
- Fully responsive (phone/tablet/PC)

## 🎯 Difficulty Ranking (Priority Order)
1. **Interchangeable exchanges** (easiest)
2. **Fewer disagreements**
3. **Closer bench rankings**
4. **More top2/bot2 agreement**
5. **Later exchanges** (hardest)

## 🚀 Live Demo
[jentaruno.github.io/bp-easy-chair](https://jentaruno.github.io/bp-easy-chair)

## 📱 Usage
1. Enter judges' calls
2. Mark interchangeable exchanges
3. Click "Calculate"
4. Resolve recommended exchanges
5. See consensus!
